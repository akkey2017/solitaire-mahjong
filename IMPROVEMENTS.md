# 一人麻雀ゲームの改善について

## 実装した機能

### 1. ゲーム性の改善
- **目標点数システム**: 単に牌を引き切るだけでなく、目標点数を目指すゲームに変更
- **難易度設定**: 3つの難易度（簡単: 500点、普通: 1000点、難しい: 2000点）から選択可能
- **流局処理**: 山が尽きたら流局として処理し、次のゲームに進む
- **勝利条件**: 目標点数に到達するとクリア

### 2. UI/UX改善
- **統計情報の表示**: 
  - 現在のスコアと目標点数
  - 勝率（勝利数/総ゲーム数）
  - 最高得点と累計得点
  - 連勝記録
  - 進捗バー（目標達成までの進捗を視覚化）

- **設定モーダル**: ゲーム難易度の変更が可能

- **ヘルプモーダル**: ゲームルールと操作方法の説明

- **勝利モーダル**: 目標達成時に表示される祝福画面
  - 続けるオプション（さらに高得点を目指す）
  - 新しく始めるオプション

- **データ永続化**: LocalStorageを使用して統計情報と設定を保存

### 3. 視覚的改善
- **アニメーション**: フェードイン、スライドインアニメーション
- **ホバーエフェクト**: ボタンやカードにホバー時の視覚的フィードバック
- **プログレスバー**: 目標達成までの進捗を視覚的に表示
- **カラーコーディング**: 統計情報ごとに異なる色を使用して視認性向上

## 技術的な改善点

1. **TypeScript型定義の拡張**: GameStatistics型とGameSettings型の追加
2. **状態管理の改善**: localStorage との連携、lazy initializer パターンの使用
3. **コンポーネント分離**: 再利用可能なモーダルコンポーネントの作成
4. **アクセシビリティ**: 適切なARIAラベルとセマンティックHTML

## 使い方

1. **ゲームを開始**: 「新しく始める」ボタンをクリック
2. **ゲームプレイ**: 牌を引いて、捨てて、役を作る
3. **和了**: 手牌が完成したら「ツモ」ボタンをクリック
4. **目標達成**: 目標点数に到達すると勝利モーダルが表示される
